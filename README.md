# hexo-blog-tool

一个可以编写hexo博客的工具，可以根据hexo配置中的地址进行推送。

包含一个md编辑器

使用hexo-deploy-tool插件自动推送到指定的仓库

目前只实现了基础功能，还有很多的bug和问题，有时间会慢慢的完善和修复，欢迎提issues和fork。

## Project setup
```
yarn install
```
